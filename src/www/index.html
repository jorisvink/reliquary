+ Introduction +

The Reliquary is a shrouded sanctum cathedral infrastructure which helps you establish PQ-secure P2P E2EE sanctum tunnels between any of your devices, even when behind NAT.

The Reliquary provides an API which interacts with our cathedrals and allows you to easily manage your flocks and devices using a handful of very simple shell scripts.

Any sanctum based application will work with this cathedral network, be it sanctum itself or libkyrka based applications such as: tier6, confessions and gospel

Check out our getting started guide to get your first two devices connected to each other.

This community service is free for use as long as you adhere to our strict nothing-illegal policy. If you prefer running this yourself, the entire reliquary source is available on our github mirror here.

+ Do you have access to my traffic? +

No.

The cathedrals cannot inject, modify or read any of your traffic that is relayed over them because they lack the keys to be able to do so, you own those keys.
